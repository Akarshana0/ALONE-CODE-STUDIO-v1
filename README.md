# ALONE CODE STUDIO v11

**Professional offline IDE — iOS · Android · Windows · Linux · PWA**

## What's New in v11
- ✅ Zero "No Internet" error screens — full offline-first PWA
- ✅ Custom icon on all platforms (Windows taskbar, Android, iOS)
- ✅ Enhanced Find & Replace — Regex / Match Case / Whole Word / ↑↓ / count
- ✅ Rename Symbol (F2) — rename all occurrences at once
- ✅ Replace All button
- ✅ Code Folding (click gutter arrows)
- ✅ Import file preview — see code before importing
- ✅ Toast notifications stack
- ✅ All 20 language offline engines (from v7)

## Run Locally
```bash
npx serve .
# or
python3 -m http.server 8080
```
Open: http://localhost:8080

## Deploy to GitHub Pages
1. Push folder to GitHub repo
2. Settings → Pages → Source: main / root
3. Done ✓

## Keyboard Shortcuts
| Shortcut | Action |
|---|---|
| Ctrl+Enter | Run code |
| Ctrl+S | Save |
| Ctrl+F | Find & Replace |
| F2 / Ctrl+R | Rename Symbol |
| Ctrl+B | Toggle Sidebar |
| Ctrl+\\ | Split View |
| Ctrl+P | Preview/Output |
| Ctrl+I | Import Files |
| Ctrl+/ | Toggle Comment |
| Escape | Close panels |

## Terminal Commands
`run` `clear` `ls` `find <text>` `rename` `new <lang>` `font <size>` `wrap` `theme <name>` `import` `download` `help`

## Install as PWA
- **iOS:** Safari → Share → "Add to Home Screen"
- **Android:** Chrome → ⋮ → "Install App"
- **Windows/Mac:** Chrome/Edge → address bar → Install ⊕
- **Linux:** Chrome → ⋮ → "Install ALONE CODE STUDIO"
